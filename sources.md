# Data sources

p3k14c is an ensemble of a great many source datasets, detailed below. We offer
our sincerest thanks and gratitude to each contributing author, without whom 
p3k14c could not be possible.

[Download this list as a BiBTeX .bib file](/data/p3k14c_refs.bib).
